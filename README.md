# React Cloudinary File Uploader

A simple React application that allows users to upload images to **Cloudinary** and displays the uploaded image URL.

---

## Features

- Upload single image files to Cloudinary
- Shows upload progress (`Uploading...`)
- Displays the uploaded image after successful upload
- Simple and minimal UI
